MeshUtility is a collection of Python utilities for triangular mesh processing, with performance-critical algorithms implemented as C++ extensions via pybind11. It supports geodesic distance computation, isocurve extraction, incremental remeshing, mesh cutting and splitting, graph construction, and visualization helpers.

## Key features

<CardGroup cols={2}>
  <Card title="Geodesic computation" icon="route">
    Exact geodesic paths and distance fields via the MMP algorithm. Fast marching with optional per-face speed weighting.
  </Card>
  <Card title="Isocurve extraction" icon="wave-sine">
    Extract isolines at any scalar value from a field defined on mesh vertices, in both Python and C++.
  </Card>
  <Card title="Incremental remeshing" icon="grid">
    Isotropic remeshing with target edge length control, feature preservation, and fine-grained step-by-step access.
  </Card>
  <Card title="Mesh cutting and splitting" icon="scissors">
    Cut meshes along vertex chains and split edges by inserting new points at arbitrary positions.
  </Card>
  <Card title="Graph utilities" icon="diagram-project">
    Build face-face and vertex-vertex adjacency graphs backed by NetworkX for connectivity queries.
  </Card>
  <Card title="Sphere CVT sampling" icon="circle-dot">
    Uniformly distribute points on the unit sphere using centroidal Voronoi tessellation.
  </Card>
</CardGroup>

## Get started in seconds

<Steps>
  <Step title="Install MeshUtility">
    ```bash
    pip install meshutility==0.0.2
    ```
  </Step>
  <Step title="Load a mesh">
    ```python
    import openmesh as om
    mesh = om.read_trimesh("my_mesh.obj")
    V = mesh.points()
    F = mesh.face_vertex_indices()
    ```
  </Step>
  <Step title="Compute geodesic distances">
    ```python
    import meshutility as mu
    import numpy as np

    source_indices = np.array([0])
    D = mu.pygeodesic.distance_field(V, F, source_indices)
    print("Geodesic distances computed:", D.shape)
    ```
  </Step>
  <Step title="Explore more">
    Check the [Guides](/guides/geodesic) for detailed walkthroughs of each feature, or jump straight to the [API Reference](/api/pygeodesic).
  </Step>
</Steps>

<Note>
  MeshUtility works with triangular meshes. Inputs are typically NumPy arrays: vertices as an `(m, 3)` float array and faces as an `(n, 3)` int array.
</Note>
