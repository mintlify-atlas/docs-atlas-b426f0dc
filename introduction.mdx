MeshUtility is a collection of Python utilities for triangular mesh processing. Algorithms that require performance are implemented as compiled C++ extensions and exposed to Python via pybind11, so you get the convenience of a Python API with near-native execution speed for the heavy lifting.

## Features

**Python modules**

- `colormap_vertex_color` ŌĆö assign per-vertex colors to visualize a scalar field on a mesh
- `cut_along_curve` ŌĆö cut a mesh along a vertex chain
- `ff_graph` / `vv_graph` ŌĆö build face-face and vertex-vertex adjacency graphs backed by NetworkX
- `get_scalar_field_on_resampled_points` ŌĆö interpolate a scalar field defined on mesh vertices onto resampled points
- `IsoCurve` ŌĆö extract isocurves from a scalar field defined on a manifold triangular mesh
- `read_obj_lines` / `write_obj_lines` ŌĆö read and write polylines from Wavefront `.obj` files
- `remove_unreferenced_vertices` ŌĆö clean up meshes by removing vertices not referenced by any face
- `sphere_cvt` ŌĆö approximate centroidal Voronoi tessellation (CVT) on the unit sphere for uniform point sampling
- `split_connected_components` ŌĆö split a mesh into its connected components
- `split_mesh` / `split_mesh_complete` ŌĆö split a mesh by inserting new vertices on edges

**C++ extension modules**

| Module | Description |
|---|---|
| `pygeodesic` | Exact geodesic distances via the MMP algorithm (Danil Kirsanov's implementation) and fast marching with optional per-face speed weighting |
| `pyisocurve` | C++ implementation of isocurve extraction, equivalent to `IsoCurve` |
| `pyremesh` | Incremental isotropic remeshing with target edge length control |
| `pyshapeop` | Partial Python binding of the ShapeOp geometric constraint solver |

## Dependencies

MeshUtility requires the following Python packages:

- [numpy](https://numpy.org)
- [scipy](https://scipy.org)
- [networkx](https://networkx.org)
- [matplotlib](https://matplotlib.org)
- [openmesh](https://www.openmesh.org/media/Documentations/OpenMesh-Doc-Latest/a04088.html)

The C++ extension modules (`pygeodesic`, `pyisocurve`, `pyremesh`, `pyshapeop`) are pre-compiled and included in the wheel ŌĆö no C++ toolchain is required when installing from PyPI.
