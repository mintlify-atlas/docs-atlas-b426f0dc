---
title: "Quickstart"
description: "Load a mesh, compute a geodesic distance field, extract an isocurve, and run isotropic remeshing."
---

This guide walks you through the three most common MeshUtility workflows: geodesic distance fields, isocurve extraction, and remeshing. By the end you will have run all three on a real mesh.

## Prerequisites

Install MeshUtility before starting:

```bash
pip install meshutility==0.0.2
```

See [Installation](/installation) if you run into any issues.

---

<Steps>
  <Step title="Load a mesh">
    MeshUtility expects vertices as an `(m, 3)` float NumPy array and faces as an `(n, 3)` int NumPy array. The easiest way to get these is via `openmesh`:

    ```python
    import numpy as np
    import openmesh as om
    import meshutility as mu

    mesh = om.read_trimesh("my_mesh.obj")
    V = mesh.points()              # (m, 3) float64
    F = mesh.face_vertex_indices() # (n, 3) int32
    ```

    <Note>
      MeshUtility only supports triangular meshes. If your mesh contains quads or polygons, triangulate it before loading.
    </Note>
  </Step>

  <Step title="Compute a geodesic distance field">
    `mu.pygeodesic.distance_field` computes exact geodesic distances from one or more source vertices to every other vertex on the mesh using the MMP algorithm.

    ```python
    # Compute geodesic distances from vertex 0
    source_indices = np.array([0])
    D = mu.pygeodesic.distance_field(V, F, source_indices)
    # D is an (m,) array of geodesic distances
    print("Max geodesic distance:", D.max())
    ```

    **Signature:**

    ```python
    mu.pygeodesic.distance_field(
        V,                           # (m, 3) float — mesh vertices
        F,                           # (n, 3) int   — mesh faces
        source_indices,              # (k,) int      — source vertex indices
        edge_split=-1.0,             # max edge length before splitting (-1 = no split)
        max_propagation_distance=-1.0  # stop propagating beyond this distance (-1 = full mesh)
    )
    ```

    <Tip>
      Pass multiple source indices to compute the distance to the nearest source from every vertex. This is useful for Voronoi-style partitioning on surfaces.
    </Tip>
  </Step>

  <Step title="Extract an isocurve">
    With a scalar field on the mesh (such as the geodesic distance field `D` computed above), you can extract the isocurve at any value using `mu.IsoCurve`.

    ```python
    # Extract the isocurve at half the maximum geodesic distance
    iso_value = D.max() / 2.0

    iso = mu.IsoCurve(V, F, D)
    pts, on_edge, ratio, curve_all = iso.extract(iso_value, eqlTol=1e-6)

    print(f"Isocurve has {len(pts)} points across {len(curve_all)} curve(s)")
    ```

    **What the return values mean:**

    | Variable | Shape | Description |
    |---|---|---|
    | `pts` | `(p, 3)` | 3D coordinates of points on the isocurve |
    | `on_edge` | `(p, 2)` | Vertex indices of the edge each point lies on |
    | `ratio` | `(p,)` | Barycentric position along each edge |
    | `curve_all` | list of lists | Ordered point indices forming each connected curve segment |

    **Signature:**

    ```python
    iso = mu.IsoCurve(
        points,  # (m, 3) float — mesh vertices
        faces,   # (n, 3) int   — mesh faces
        field    # (m,) float   — scalar field values at vertices
    )
    pts, on_edge, ratio, curve_all = iso.extract(
        val,          # float — the iso-value to extract
        eqlTol=1e-6   # float — tolerance for treating a vertex as lying on the isocurve
    )
    ```
  </Step>

  <Step title="Remesh to a uniform edge length">
    `mu.pyremesh.remesh` applies incremental isotropic remeshing, iteratively splitting long edges, collapsing short edges, flipping edges to improve valence, and smoothing vertices until the average edge length converges toward `targetEdgeLength`.

    ```python
    target_edge_length = 0.05  # choose a value appropriate for your mesh scale

    V1, F1 = mu.pyremesh.remesh(V, F, target_edge_length)

    print(f"Original mesh: {len(V)} vertices, {len(F)} faces")
    print(f"Remeshed:      {len(V1)} vertices, {len(F1)} faces")
    ```

    **Signature:**

    ```python
    V1, F1 = mu.pyremesh.remesh(
        V,                # (m, 3) float — input vertices
        F,                # (n, 3) int   — input faces
        targetEdgeLength  # float        — desired edge length after remeshing
    )
    ```

    <Tip>
      A good starting value for `targetEdgeLength` is roughly the average edge length of your input mesh. You can estimate it as `np.linalg.norm(V[F[:, 0]] - V[F[:, 1]], axis=1).mean()`.
    </Tip>
  </Step>
</Steps>
